Общая информация

Эта програма для просмотра информации Московских безконтактных билетов. Она работает только с "бумажными" билетами. Можно увидеть пригодность билета, время действия, как много осталось поездок. Также можно посмотреть информацию о использованой в билете микросхеме. Можно отослать дамп по E-Mail. Исходные коды и свежая версия здесь

Режимы работы

Эта програма может показать информацию о билете полученую с помощью NFC или использовать сохранённый дамп переданый другой програмой (например OI File Manager или X-plore File manager)

Limitations

  • Програма не требует прав доступа на чтение или запись. Она может работать только с файлами в этом каталоге и ниже:
    Это каталог выделенный специально для этой программы. При запуске файлового менеджера из этого приложения этот каталог открывается автоматически. Внимание! При удалении программы все данные из этого каталога будут стёрты!
  • Приложение работает только с "бумажными" билетами на основе микросхем: Mifare®, Mifare Ultralight®, микросхемы ОАО Микрон. Это означает что прогограмма не работает с билетами на основе Mifare 1K®. Это билеты Тройка, Стрелка и другие "пластиковые" билеты.

Settings

  • Язык программы
    • Установить язык программы. Можно использовать Английский, Русский или язык установленный в системе.
  • Отправлять информацию об устройстве вместе с дампом
    • Отправлять общую информацию об устройстве и служебную информацию из программы вместе с дампом
  • Использовать VIEW_DIRECTORY
    • Использовать разные способы запуска файлового менеджера. Некоторые файловые менеджеры (X-plore File Manager, ES File Manager) требуют включения этой опции. Другие (OI File Manager) требуют выключения этой опции.
