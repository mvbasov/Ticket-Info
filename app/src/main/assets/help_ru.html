Общая информация

Эта програма для просмотра информации Московских безконтактных билетов. Она работнает только с "бумажными" билетами Можно увидеть пригодность билета, время действия, как много осталось поездок. Также можно посмотреть информацию о использованой в билете микросхеме. Можно отослать дамп по E-Mail

Режимы работы

Эта програма может показать информацию о билете полученую с помощью NFC или использовать сохранённый дамп переданый другой програмой (например OI File Manager или XPlore File manager)
