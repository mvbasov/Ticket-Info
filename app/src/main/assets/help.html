Ticket info

To change settings press button below:

General

This program for view information from Moscow transportation system contactless tickets. It operate only with "paper" tickets. You can see ticket state, valid time perion, how many trips it has. Also application can show information about used IC. You can send dump by E-Mail also.

Operation modes

This program display ticket content scanned by NFC or show saved dump shared by another application (OI File Manager or XPlore file manager as example)
