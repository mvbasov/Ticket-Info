General

This program for view information from Moscow transportation system contactless tickets. You can see ticket state, valid time perion, how many trips it has. Also application can show information about used IC. You can send dump by E-Mail also. Source and latest version here

Operation modes

This program display ticket content scanned by NFC or show saved dump shared by another application (OI File Manager or X-plore file manager as example)

Limitations

  • Program has no Android permissions to read/write storage. It can operate with files stored only there and below:
    This is place dedicated for this application.
    When File Manager launched from this application by pressing button this directory opens automaticaly. Warning! When application deleted all files from this catalog deleted too!
  • It operate only with "paper" tickets based on ICs: Mifare®, Mifare Ultralight®, several JSC Micron ICs. It mean not operate with Mifare 1K®. So, Troika, Strelka and onther "plastic" cards can't be read.

Settings

  • Interface language
    • Set interface language. You can set English, Russian or use language set on system.
  • Transliterate stations names
    • Transliterate stations name or display its original Russian names
  • Send device info with dump
    • Send some general information about device and internal application information with dump.
  • Use VIEW_DIRECTORY
    • Use different ways to launch File manager from application. Some File Managers (X-plore File Manager, ES File Manager) require this option to be switched on. Another (OI File Manager) require option to be switched off.
